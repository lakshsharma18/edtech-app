ED-TECH PLATFORM KNOWLEDGE BASE

GENERAL OVERVIEW
ED-TECH is a comprehensive online learning platform built to provide accessible education post-COVID. It offers pre-recorded video sessions, PDF notes, and specialized courses for students and professionals.

USER ACCOUNTS & AUTHENTICATION
- Registration: Users can create an account by providing their First Name, Last Name, Email, and a secure Password.
- Login Methods: Users can log in using their Email and Password or via a secure 4-digit OTP (One-Time Password) sent to their registered email.
- Roles: The platform has two roles: "Student" (default) and "Admin".
- Security: All sessions are secured using JWT (JSON Web Tokens). If a session expires, the user must log in again.

STUDENT FEATURES
- Course Discovery: Students can browse all available courses on the "Courses" page.
- Course Details: Clicking on a course allows students to see the description, price, and the curriculum (list of lessons).
- Learning: Once enrolled, students can watch high-quality videos and view/download PDF notes for each lesson.
- Account Management: Students can see their name and profile status in the top-right navbar dropdown.

ADMIN FEATURES (INSTRUCTORS)
- Admin Dashboard: Only users with the "Admin" role can access the Admin Panel at /admin/dashboard.
- Course Management: Admins can Create, Read, Update, and Delete (CRUD) courses.
- Lesson Management: Admins can add lessons to courses. Each lesson must have a title and a video file (MP4). PDF notes are optional.
- Video Storage: All videos and PDFs are securely stored in Azure Blob Storage.
- Course Deletion: If an admin deletes the last remaining lesson of a course, the entire course is automatically removed from the database to maintain platform quality.

TECHNICAL SPECIFICATIONS
- Frontend: Built with React and TypeScript using Bootstrap 5 for a responsive, modern UI.
- Backend: Powered by FastAPI for high-performance API routing.
- Database: PostgreSQL is used for storing user data, course info, and lesson metadata.
- Cloud Storage: Azure Blob Storage is used for hosting all educational media assets.
- Animations: Powered by Framer Motion for smooth transitions and interactive elements.

SUPPORT & CONTACT

- The platform is available 24/7 for global learners.
